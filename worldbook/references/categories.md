# 世界书分类体系与写作指南

## 分类一览

| 分类 | 适用内容 | 优先级建议 | 示例 |
|------|----------|------------|------|
| 地点 | 城市、建筑、地貌、国家、区域、地下城 | 10-20 | 暴风城、死亡矿井、艾尔文森林 |
| 人物 | 重要 NPC、历史人物、传说角色 | 20-30 | 国王瓦里安、大法师安东尼达斯、商人哈罗德 |
| 组织 | 公会、王国、教派、地下组织、商会 | 15-25 | 冒险者公会、血色十字军、暗影议会 |
| 历史 | 战争、纪元、传说、重要事件 | 5-15 | 第一次大战、黑暗之门开启、上古之神封印 |
| 魔法 | 法术体系、魔法物品、魔法材料 | 15-30 | 奥术魔法、符文铭刻、元素契约 |
| 种族 | 精灵、矮人、兽人、龙族等 | 10-20 | 高等精灵、山丘矮人、半兽人 |
| 物品 | 神器、传说武器、特殊道具 | 20-40 | 霜之哀伤、龙之宝珠、贤者之石 |
| 地理 | 国家、大陆、地形地貌、气候带 | 10-20 | 艾泽拉斯大陆、无尽之海、贫瘠之地 |
| 力量体系 | 超自然能力、修炼等级、能量来源 | 15-30 | 星辉、灵力、查克拉、霸气 |
| 文化 | 货币体系、法律、习俗、节日 | 5-15 | 月轮历法、丰收祭、金币铜币银币 |
| 知识 | 技术、学术、秘传知识 | 10-20 | 炼金术配方、上古文字、机械构造 |
| 其他 | 以上分类之外的杂项 | 5-15 | 未分类条目 |

## 优先级 (order) 使用指南

- **5-15**：背景信息、历史事件、通用知识（低频触发不碍事）
- **16-25**：常见地点、主要组织、核心种族（常常用到）
- **26-40**：当前剧情核心人物、神器、关键物品（高频优先显示）

## 条目编写原则

### 简洁

每个条目 2-5 句话。AI 的上下文窗口有限，世界书条目过长会挤占对话空间。

坏例子（太长）：
> 暴风城是人类王国暴风王国的首都，坐落于艾尔文森林的北部海岸。这座城市由暴风王国的开国君主洛萨一世在黑暗之门开启前 1200 年建立，城墙由当地采石场开采的白色花岗岩建造，因此得名"白城"。暴风城经历了三次兽人战争，第一次战争中曾被兽人部落攻陷并焚毁，后在第二次战争后由石匠工会重建...

好例子（精炼）：
> 暴风城是人类最繁华的港口都城，白花岗岩城墙高耸。城内有贸易区、法师区、旧城区和矮人区。现任国王安度因·乌瑞恩。狮鹫栖于城墙塔楼，是城市防御的空中力量。

### 自包含

条目内容独立完整，不依赖其他条目才能理解。不要写"详见某某条目"。

### 关键词覆盖

关键词要有多个变体，覆盖 AI 和玩家可能使用的称呼：

- 中文全名：艾尔文森林
- 简称/昵称：艾尔文、新手森林
- 英文/原名：Elwynn Forest
- 相关词：暴风城郊外、西部荒野边界
- 常见拼写变体：Elwyn, Elwynn

### 聚焦 AI 需要的信息

世界书不是 Wiki——只写对角色扮演有用的信息：
- **要写**：氛围、常见互动、NPC 态度、已知危险、商机
- **不写**：无关的历史细节、精确的坐标数据、游戏机制数值

## 条目模板

### 地点

```
标题: [地点名]
分类: 地点
关键词: [全名], [简称], [英文名], [附近地标]
内容: [氛围描述 1-2 句]。[主要特征 1-2 句]。[NPC/住民/商人]。[危险/机遇，如有]。
优先级: 10-20
```

**示例：**
```
标题: 冒险者公会大厅
关键词: 冒险者公会, 公会大厅, 任务板, guild
内容: 冒险者公会大厅位于城市中心广场旁，是一栋三层石木结构建筑。一楼是任务发布大厅，墙上挂满了悬赏告示。接待员艾琳负责登记冒险者信息并发放任务。公会提供基础的武器修理和药水补给服务。
优先级: 20
```

### 人物

```
标题: [人物名]
分类: 人物
关键词: [全名], [称号], [昵称], [身份]
内容: [身份+外貌 1 句]。[性格特征 1 句]。[对冒险者的态度 1 句]。[已知的秘密/特殊能力，如有]。
优先级: 20-30
```

### 组织

```
标题: [组织名]
分类: 组织
关键词: [全名], [简称], [成员称呼]
内容: [组织性质+规模 1 句]。[主要活动/目标 1 句]。[对冒险者的态度 1 句]。[标志/识别方式，如有]。
优先级: 15-25
```

## 批量创建示例

```bash
# 创建世界书
python scripts/wb.py init my-fantasy-world

# 批量添加条目（复制以下命令替换内容即可）
python scripts/wb.py create-entry stormwind --title "暴风城" --keywords "暴风城,Stormwind,人类都城,白城" --category "地点" --content "暴风城是人类最繁华的港口都城，白花岗岩城墙高耸。城内有贸易区、法师区、旧城区和矮人区。现任国王安度因·乌瑞恩。狮鹫栖于城墙塔楼。"

python scripts/wb.py create-entry erin --title "艾琳" --keywords "艾琳,Erin,公会接待员,冒险者公会" --category "人物" --content "艾琳是冒险者公会的接待员，25岁左右的人类女性，棕发碧眼。性格开朗热情，对新手冒险者特别照顾。负责登记任务、发放报酬和维护任务板。"

python scripts/wb.py create-entry adventurers-guild --title "冒险者公会" --keywords "冒险者公会,公会,guild,冒险者" --category "组织" --content "冒险者公会是横跨大陆的独立组织，不隶属于任何王国。为冒险者提供任务中介、信息交换和基础补给服务。公会徽章是一把交叉的剑和羽毛笔，象征行动与知识的结合。"
```
