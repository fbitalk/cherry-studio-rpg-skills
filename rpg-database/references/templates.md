# RPG 表格模板参考

以下是游戏数据库的预定义表格模板。使用 `python scripts/db.py create-table` 创建表格，引用对应的列头。

---

## 模板速查

| 模板键 | 中文名 | 列头 | 使用场景 |
|--------|--------|------|----------|
| `global-state` | 全局状态 | 角色名, 当前地点, 当前时间, 上轮时间, 经过时间, 备注 | 追踪时间线、地点变化 |
| `protagonist-info` | 主角信息 | 角色名, 性别/年龄, 外貌特征, 当前衣着, 职业/身份, 备注 | 主角核心身份档案 |
| `character-stats` | 角色属性 | 角色名, 属性名, 当前值, 最大值, 备注 | 跟踪 HP、MP、力量等数值属性 |
| `inventory` | 背包物品 | 物品名称, 数量, 类别, 归属, 描述, 备注 | 管理多人背包/物品栏 |
| `quests` | 任务日志 | 任务名称, 任务类型, 状态, 发布者, 详细描述, 当前进度, 任务时限, 奖励, 惩罚 | 追踪任务完整生命周期 |
| `npc-relations` | 角色关系 | 姓名, 性别/年龄, 外貌特征, 当前衣着, 身份/职业, 好感度, 是否在场, 背景故事, 关键经历, 人际关系, 简介, 别称, 备注 | 深度 NPC 档案与关系网 |
| `skills` | 技能列表 | 技能名称, 技能类型, 等级, 消耗, 效果描述 | 管理角色技能与魔法 |
| `equipment` | 装备栏 | 装备名, 部位, 属性加成, 耐久度, 归属, 描述 | 追踪多人装备状态 |
| `memo` | 备忘录 | 标题, 分类, 归档状态, 记录日期, 记录内容, 涉及对象, 关键要素 | 线索/成就/约定追踪 |
| `factions` | 势力表 | 名称, 描述, 领袖, 关系, 位置 | 管理组织与势力 |
| `locations` | 地点表 | 地点名称, 所属区域, 地点描述, 地点现状 | 重要地点档案 |
| `chronicle` | 纪要表 | 时间跨度, 地点, 纪要, 概览, 编码索引 | 每轮剧情事件日志 |

## 1. 全局状态 (global-state)

追踪游戏世界的时间线、主角当前位置。此表通常只有 1 行（单主角时）。

**创建命令：**
```bash
python scripts/db.py create-table global-state --headers "角色名,当前地点,当前时间,上轮时间,经过时间,备注"
```

**列说明：**
| 列 | 说明 |
|----|------|
| 角色名 | 主角名称 |
| 当前地点 | 主角当前所在的场景名称 |
| 当前时间 | 游戏世界当前时间，格式 `x年x月x日 HH:MM`，初始根据世界观设定 |
| 上轮时间 | 上一轮交互结束时的时间 |
| 经过时间 | 根据当前与上轮时间计算得出的文本描述（如"约1小时"、"几分钟"） |
| 备注 | 补充说明 |

**操作规则：**
- 每次主角切换地点时更新「当前地点」
- 每轮交互结束时更新时间和经过时间
- 禁止删除，禁止新增多行（单主角）

**示例数据：**

| 角色名 | 当前地点 | 当前时间 | 上轮时间 | 经过时间 | 备注 |
|--------|----------|----------|----------|----------|------|
| 艾伦 | 冒险者公会大厅 | 星历1250年霜月3日 14:30 | 星历1250年霜月3日 08:00 | 约6小时 | |

**插入示例：**
```bash
python scripts/db.py insert-row global-state --data '{"角色名":"艾伦","当前地点":"冒险者公会大厅","当前时间":"星历1250年霜月3日 14:30","上轮时间":"","经过时间":"0分","备注":""}'
```

## 2. 主角信息 (protagonist-info)

记录主角的核心身份信息。此表通常只有 1 行。

**创建命令：**
```bash
python scripts/db.py create-table protagonist-info --headers "角色名,性别/年龄,外貌特征,当前衣着,职业/身份,备注"
```

**列说明：**
| 列 | 说明 |
|----|------|
| 角色名 | 主角的名字 |
| 性别/年龄 | 生理性别和年龄，格式如 `男/22` |
| 外貌特征 | 包含发型、发色、身高、身材、肤色、五官。不含神态描写、姿势或临时状态。低频更新 |
| 当前衣着 | 当前服装搭配。低频更新，换装或衣物破损时更新 |
| 职业/身份 | 主角在社会中的主要角色 |
| 备注 | 补充信息 |

**操作规则：**
- 禁止删除
- 外貌特征仅在重大改变时更新
- 衣着在换装/破损时更新

**示例数据：**

| 角色名 | 性别/年龄 | 外貌特征 | 当前衣着 | 职业/身份 | 备注 |
|--------|-----------|----------|----------|-----------|------|
| 艾伦 | 男/22 | 深棕色短发，身高178cm，体型中等，小麦色皮肤，五官轮廓分明 | 深蓝旅行斗篷、灰色布制上衣、黑色长裤、皮革长靴 | 冒险者 | |

**插入示例：**
```bash
python scripts/db.py insert-row protagonist-info --data '{"角色名":"艾伦","性别/年龄":"男/22","外貌特征":"深棕色短发，身高178cm，体型中等，小麦色皮肤","当前衣着":"深蓝旅行斗篷、灰色布制上衣、黑色长裤、皮革长靴","职业/身份":"冒险者","备注":""}'
```

## 5. 任务日志 (quests)

追踪所有任务的完整生命周期。

**创建命令：**
```bash
python scripts/db.py create-table quests --headers "任务名称,任务类型,状态,发布者,详细描述,当前进度,任务时限,奖励,惩罚"
```

**列说明：**
| 列 | 说明 |
|----|------|
| 任务名称 | 任务标题 |
| 任务类型 | 主线任务 / 支线任务 |
| 状态 | 未开始 / 进行中 / 已完成 / 已失败 |
| 发布者 | 发布该任务的角色或势力 |
| 详细描述 | 任务的目标和要求 |
| 当前进度 | 对完成度的简要描述 |
| 任务时限 | 剩余时间，无限制填"无" |
| 奖励 | 完成可获得的奖励 |
| 惩罚 | 失败的后果，无惩罚填"无" |

**操作规则：**
- 完成任务 → 更新状态为「已完成」
- 任务失败/过期 → 更新状态为「已失败」，不要删除
- 取得关键进展 → 更新「当前进度」

**示例数据：**

| 任务名称 | 任务类型 | 状态 | 发布者 | 详细描述 | 当前进度 | 任务时限 | 奖励 | 惩罚 |
|----------|----------|------|--------|----------|----------|----------|------|------|
| 消灭哥布林 | 支线任务 | 进行中 | 村长 | 清除森林东侧洞穴中的哥布林 | 已找到洞穴入口 | 3天内 | 50金币 | 声望-10 |
| 寻找魔法水晶 | 主线任务 | 未开始 | 法师塔 | 深入地下城第三层取回魔法水晶 | 无 | 无 | 魔法戒指 | 无 |
| 护送商队 | 支线任务 | 已完成 | 商人工会 | 护送商队安全通过峡谷 | 已安全抵达 | 无 | 200金币 | 赔偿损失 |

**插入示例：**
```bash
python scripts/db.py insert-row quests --data '{"任务名称":"消灭哥布林","任务类型":"支线任务","状态":"进行中","发布者":"村长","详细描述":"清除森林东侧洞穴中的哥布林","当前进度":"已找到洞穴入口","任务时限":"3天内","奖励":"50金币","惩罚":"声望-10"}'
```

## 6. 角色关系 (npc-relations)

深度记录 NPC 档案与关系网。每个重要 NPC 一行。

```bash
python scripts/db.py create-table npc-relations --headers "姓名,性别/年龄,外貌特征,当前衣着,身份/职业,好感度,是否在场,背景故事,关键经历,人际关系,简介,别称,备注"
```

| 列 | 说明 |
|----|------|
| 姓名 | NPC 名字 |
| 性别/年龄 | 格式如 `女/25` |
| 外貌特征 | 发型、发色、身高、身材、肤色、五官。不含神态/姿势/临时状态。重大改变时才更新 |

| 是否在场 | 是/否，角色是否已离开当前剧情 |

## 9. 备忘录 (memo)

记录主角获得的线索、成就、约定等。不是流水账，只在事件闭环结束后汇总记录。

**创建命令：**
```bash
python scripts/db.py create-table memo --headers "标题,分类,归档状态,记录日期,记录内容,涉及对象,关键要素"
```

**列说明：**
| 列 | 说明 |
|----|------|
| 标题 | 备忘录标题，内容的概括 |
| 分类 | 秘密线索类 / 大事件类 / 约定承诺类 |
| 归档状态 | 未归档 / 已归档 / 已废除 |
| 记录日期 | 游戏内日期，格式 `x年x月x日 HH:MM` |
| 记录内容 | 客观详细的事件记录，不升华不评价 |
| 涉及对象 | 涉及的角色名 |
| 关键要素 | 核心要素短语，最多2个，用于触发检索 |

**分类说明：**
- **秘密线索类**：获得关键情报、发现隐藏信息、解开谜题
- **大事件类**：完成重要任务、经历重大事件、获得成就
- **约定承诺类**：与他人达成约定、许下承诺、签订契约

**操作规则：**
- 只有事件闭环结束才添加，禁止对连续事件做流水账
- 已归档/已废除的行禁止更新
- 「归档状态」一旦设为已归档/已废除则不可回退
- 禁止删除已有条目

**示例数据：**

| 标题 | 分类 | 归档状态 | 记录日期 | 记录内容 | 涉及对象 | 关键要素 |
|------|------|----------|----------|----------|----------|----------|
| 发现哥布林巢穴 | 秘密线索类 | 未归档 | 星历1250年霜月3日 10:00 | 在森林东侧发现了哥布林的洞穴入口，门口有哨兵巡逻 | 艾伦 | 哥布林巢穴 |
| 加入冒险者公会 | 大事件类 | 已归档 | 星历1250年霜月3日 08:30 | 在冒险者公会完成登记，正式成为冒险者，获得了冒险者徽章 | 艾伦、艾琳 | 冒险者公会 |

**插入示例：**
```bash
python scripts/db.py insert-row memo --data '{"标题":"发现哥布林巢穴","分类":"秘密线索类","归档状态":"未归档","记录日期":"星历1250年霜月3日 10:00","记录内容":"在森林东侧发现了哥布林的洞穴入口，门口有哨兵巡逻","涉及对象":"艾伦","关键要素":"哥布林巢穴"}'
```


**列说明：**

|----|------|

| 描述 | 势力性质、宗旨、组织结构 |
| 领袖 | 当前领导者 |

| 位置 | 大本营或主要活动区域 |

**操作规则：**
- 势力覆灭/解散时删除
- 领袖、关系变动时更新


## 11. 地点表 (locations)

| 列 | 说明 |
|----|------|
| 地点名称 | 地点名称 |

## 12. 纪要表 (chronicle)

每轮游戏交互的剧情日志。每轮结束后追加一条。

**创建命令：**
```bash
python scripts/db.py create-table chronicle --headers "时间跨度,地点,纪要,概览,编码索引"
```

**列说明：**
| 列 | 说明 |
|----|------|
| 时间跨度 | 本轮事件发生的时间范围，格式 `开始时间 ~ 结束时间` |
| 地点 | 本轮事件发生的地点 |
| 纪要 | 按时间顺序客观记录发生的核心事件，抓主放次，不做评价 |
| 概览 | 30字以内，2-3句话概括 |
| 编码索引 | 格式 `AMXXXX`，从 `AM0001` 递增，唯一 |

**操作规则：**
- 每轮交互结束后插入一条
- 禁止删除
- 禁止更新已有记录（只追加，不修改）

**示例数据：**

| 时间跨度 | 地点 | 纪要 | 概览 | 编码索引 |
|----------|------|------|------|----------|
| 星历1250年霜月3日 08:00 ~ 08:30 | 主角的家→冒险者公会 | 艾伦从家中出发前往冒险者公会。在公会大厅遇到了接待员艾琳，完成了冒险者身份登记。 | 主角登记成为冒险者，结识接待员艾琳 | AM0001 |
| 星历1250年霜月3日 08:30 ~ 10:00 | 冒险者公会→东部森林 | 艾伦在任务板接取了村长的委托，前往东部森林调查哥布林。在森林中发现洞穴入口。 | 接取哥布林任务，发现洞穴入口 | AM0002 |

**插入示例：**
```bash
python scripts/db.py insert-row chronicle --data '{"时间跨度":"星历1250年霜月3日 08:00 ~ 08:30","地点":"主角的家→冒险者公会","纪要":"艾伦从家中出发前往冒险者公会。在公会大厅遇到了接待员艾琳，完成了冒险者身份登记。","概览":"主角登记成为冒险者，结识接待员艾琳","编码索引":"AM0001"}'
```

## 自定义模板

以上模板可根据需要修改列头。如果创建自定义表格，只需自行指定列头：

```bash
python scripts/db.py create-table my-table --headers "列1,列2,列3"
```

列头中不要包含半角逗号（逗号是分隔符）。如果确实需要逗号，使用中文逗号（，）代替。
